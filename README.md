# Form Survey Perokok

Aplikasi ini adalah Form Survey Perokok untuk menginput data seperti nama, jenis kelamin, dan pilihan rokok.
Data yang telah disubmit akan disimpan dan ditampilkan di halaman submission, tidak hilang saat halaman direfresh.

![Form Survey Perokok](./images/image.png)

![Submission](./images/image2.png)

## Cara Menjalankan Aplikasi

- Clone repositori ini
- Buka index.html
- Isi form survey perokok
- Klik submit, maka hasilnya akan terlihat di halaman submission
- Jika ingin mengisi data baru, klik kembali ke form
- Isi data baru
- Klik submit, maka akan mengarah ke halaman submission
- Dan data lama akan berada di urutan atas
- Jika ada data baru maka ulangi lagi proses yang sama
- Selesai
